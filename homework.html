Homework guidelines

  • The deadline hour is 23:55 on the deadline day.
  • Homework must be submitted via the Moodle system
  • Sumbit an ipynb file, file name: hwX_id.ipynb (replace X with homework number and id with your ID number). For example, the sumbission of homework 5 for a student with ID number 033545885 is hw5_033545885.ipynb
  • Submitted code must run on IPython Notebook with Python 3.4 installed with Pyzo.
  • Some of the questions are only checked automatically. Hence:
    - Do not change given function names, arguments and types.
    - Make sure that the returned values are exactly as required.
  • Grades will be returned via the Moodle system, solutions will appear on the course website (here).
  • There will be 7 exercises: 4 during the first 4 weeks and 3 during the next 6 weeks.
  • Homework grades are 0, 60, 80, 90 and 100.
  • Students must submit and pass (grade >= 60) at least 6 homeworks to complete the course.
  • The homeworks grade will contribute 20-30% of the final grade, but only if the exam grade is >= 60. The homeworks grade will be the average of the 6 best grades.
  • Homework is personal and should be solved alone. Students are not allowed to solve homework together, show solutions to other students or read other students' solutions.
  • Homework can be discussed without sharing code. Use the forum!
  • Each student has 5 days of "grace" for late submissions. For each late submission we take off a full day (or days) of "grace", even it the homework was submitted two minutes after the deadline. If a student has no more days of "grace", late submissions will not be checked and will be graded with a zero.
  • In case of a late submission, there is no need to contact staff members. If the late submission is justified, students should attach the relevant documents (sick days, reserve duty etc.) to the submission. This includes cases in which there is a malfunction with the Moodle system.
